# 🚀 SkyNet Development

Open Source ISP Management Platform

---

## Vision

SkyNet adalah platform manajemen ISP modern yang mengintegrasikan:

- CRM Pelanggan
- Billing Otomatis
- MikroTik Native Integration
- GenieACS TR-069
- WhatsApp Gateway
- Payment Gateway
- Ticketing
- Fiber GIS Map
- Monitoring
- Inventory
