# SkyNet ISP Platform Roadmap

## Vision

SkyNet adalah platform OSS/BSS untuk ISP berbasis Fiber Optic yang menggabungkan Billing, CRM, MikroTik Management, GenieACS TR-069, WhatsApp Gateway, GIS Fiber Mapping, Monitoring, dan Payment Gateway dalam satu sistem.
